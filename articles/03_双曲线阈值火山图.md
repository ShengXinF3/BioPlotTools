# 生信绘图工具 | 双曲线阈值火山图（通用版）

这是生信绘图工具系列的第三篇，主要整理一些日常分析中常用的绘图函数，方便自己用，也分享给有需要的人。

今天分享的是**双曲线阈值火山图**的可视化函数。传统的火山图使用固定的P值和效应值阈值（如P<0.05, |log2FC|>1），但这种方法可能会遗漏一些重要的基因。这个函数采用双曲线阈值（hyperbolic threshold），能更合理地识别显著基因。

## 主要功能

### 双曲线阈值自动识别显著基因

传统火山图使用固定阈值（P<0.05, |log2FC|>1），但这种方法有局限：效应值很大但P值略高于0.05的基因会被遗漏，效应值很小但P值极低的基因可能不具有生物学意义。

双曲线阈值使用公式：`|effect × -log10(P)| > threshold`，更科学合理。

```r
# 基础用法
plot_volcano_hyperbolic(
  data = data,
  threshold = 6,  # 阈值参数，通常4-8之间
  x_label = "Effect Size (AU)"
)
```

### 基因分类和高亮显示

可以为特定基因添加功能分类，并高亮显示重要基因：

```r
# 创建基因分类
gene_category <- data.frame(
  gene = c("TF1", "TF2", "ENZ1"),
  category = c("Transcription factor", "Transcription factor", "Metabolic enzyme")
)

# 绘图
plot_volcano_hyperbolic(
  data = data,
  gene_category = gene_category,
  highlight_genes = "TF1",  # 特别高亮TF1
  show_legend = TRUE
)
```

## 参数说明

主要参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| data | 数据框（必需gene, effect_size, pvalue列） | 必需 |
| gene_category | 基因分类数据框（gene, category列） | NULL |
| threshold | 双曲线阈值参数 | 6 |
| control_pattern | 对照组名称模式（如"Control"） | NULL |
| highlight_genes | 需要高亮的基因向量 | NULL |
| base_colors | 基础分组配色（命名向量） | 自动 |
| point_size | 点大小 | 2.5 |
| x_label | X轴标签 | "Effect Size" |
| plot_width | 图宽（英寸） | 6.5 |
| plot_height | 图高（英寸） | 4 |
| dpi | PNG分辨率 | 300 |
| output_name | 输出文件名（不含扩展名） | NULL |
