# 生信绘图工具 | 富集分析组合图（GO + KEGG整合展示）

这是生信绘图工具系列的第四篇，主要整理一些日常分析中常用的绘图函数，方便自己用，也分享给有需要的人。

今天分享的是**富集分析组合图**的可视化函数。传统的富集分析结果通常分开展示（GO BP一张图、GO CC一张图、KEGG一张图），占用大量版面且不便于对比。这个函数可以将多个富集分析结果（GO BP/CC/MF、KEGG等）整合到一张图中展示，**节省版面、便于对比、更加专业**。

## 先看效果

这个函数生成的图包含以下元素：

### 示例1：GO + KEGG 组合图

![GO+KEGG组合图](https://cdn.jsdelivr.net/gh/ShengXinF3/DDD@main/Pics/01_go_kegg_combined.png)

- **左侧类别标签**：彩色背景区分不同数据库（BP/CC/MF/KEGG）
- **中间基因数量气泡**：圆点大小表示富集的基因数量
- **右侧条形图**：长度表示显著性（-log10 p.adjust）
- **条形图内嵌文字**：通路描述和基因名称直接显示在条形图内

### 示例2：简洁模式（避免基因过多）

- 通路描述居中显示，更加简洁
- 基因数量通过左侧圆点展示
- 适合基因数量较多的场景

### 示例3：只展示KEGG通路

![KEGG通路](https://cdn.jsdelivr.net/gh/ShengXinF3/DDD@main/Pics/03_kegg_only.png)

- 布局自动调整

## 主要功能

### 整合多个富集分析结果

将GO（BP/CC/MF）和KEGG等多个数据库的富集结果整合到一张图中：

```r
# 基础用法
plot_combined_enrichment(
  enrichment_list = list(
    BP = ego_bp,      # GO Biological Process
    CC = ego_cc,      # GO Cellular Component
    MF = ego_mf,      # GO Molecular Function
    KEGG = ekegg      # KEGG Pathway
  ),
  top_n = 5  # 每个类别展示top 5通路
)
```

### 灵活控制基因显示方式

提供三种基因显示模式，适应不同场景：

```r
# 模式1：简洁模式（基因数量多时使用）
plot_combined_enrichment(
  enrichment_list = list(BP = ego_bp, KEGG = ekegg),
  show_genes = FALSE  # 不显示基因名，通路描述居中
)

# 模式2：完整模式（显示所有基因）
plot_combined_enrichment(
  enrichment_list = list(BP = ego_bp, KEGG = ekegg),
  max_genes = 999  # 显示所有基因，不截断
)

# 模式3：默认模式（显示部分基因）
plot_combined_enrichment(
  enrichment_list = list(BP = ego_bp, KEGG = ekegg),
  max_genes = 16  # 显示前16个基因
)
```

### 输出格式

每次运行会同时生成PDF和PNG两种格式，PDF是矢量图适合投稿，PNG适合做PPT。

## 参数说明

主要参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| enrichment_list | 命名列表，包含富集分析结果 | 必需 |
| top_n | 每个类别展示的通路数量 | 5 |
| p_cutoff | P值阈值 | 0.05 |
| show_genes | 是否显示基因名称 | TRUE |
| max_genes | 最多显示的基因数量 | 12 |
| category_colors | 类别标签配色（命名向量） | 自动 |
| bar_colors | 条形图配色（命名向量） | 自动 |
| title | 图表标题 | NULL |
| plot_width | 图宽（英寸） | 14 |
| plot_height | 图高（英寸） | 自动 |
| dpi | PNG分辨率 | 300 |
| output_name | 输出文件名（不含扩展名） | NULL |
